## :checkered_flag: Starting

```bash
# Clone this project
$ git clone https://github.com/agostinhomarcia/my-feedback.git
# Access
$ cd my-feedback
# Install dependencies
$ yarn or npm
# Run the project
$ yarn start or npm start
# The server will initialize in the <http://localhost:3000>
```
